Geno-Diver Simulation Software



Geno-Diver is simulation program designed to simulate complex traits and population structures that resemble livestock populations. The primary difference between Geno-Diver and other simulation programs is that it is able to simulate traits with either a quantitative and/or fitness genetic architecture. In addition, multiple selection protocols and mating design methods are utilized within the software. As time progresses more modules will be introduced in order to stay in line with the current models and methods used in animal breeding.
